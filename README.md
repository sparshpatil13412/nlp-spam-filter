## NLP Spam Filter
A complete machine learning pipeline for spam detection using natural language processing techniques. This project covers text preprocessing, TF-IDF feature extraction, model training with Logistic Regression, performance evaluation using precision-recall metrics, and deployment via a Streamlit web app for interactive predictions.

## Features:
  Text Preprocessing: Cleans incoming text by removing punctuation, stopwords, and normalizing text.
  
  Feature Extraction: Uses TF-IDF to convert text into meaningful numerical features for machine learning models.
  
  Model Training: Trains a Logistic Regression classifier to detect spam messages with high accuracy.
  
  Performance Evaluation: Generates classification reports, confusion matrices, and precision-recall curves.
  
  Interactive Web App: Deploys the trained model via Streamlit for real-time spam prediction.
  
  Modular Codebase: Easily replace or upgrade preprocessing, vectorization, or model components.
